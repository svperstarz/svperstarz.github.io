준비에서 실행까지단 몇 초면 됩니다.

빠른 시작 절차

~ $ gem install jekyll

~ $ jekyll new my-awesome-site

~ $ cd my-awesome-site

~/my-awesome-site $ jekyll serve

# => 브라우저로 http://localhost:4000 를 열어봅니다
