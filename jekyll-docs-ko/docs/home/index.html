환영합니다

이 사이트의 목표는 Jekyll 종합 안내서를 제공하는 것입니다. 지금부터 사이트 구성 및 실행 방법, 컨텐츠 작성 및 관리 방법, 사이트의 외관과 동작방식 수정 방법, 다양한 시스템에 전개하는 방법, Jekyll 개발에 기여하는 방법 등의 주제를 다룰 것입니다.

그래서, Jekyll 이 정확히 뭔가요?

Jekyll 은 블로그 지향적인, 아주 간단한 정적 사이트 생성기입니다. 템플릿 디렉토리에서 다양한 포맷의 원본 텍스트 파일을 읽은 후, Markdown (또는 Textile) 과 Liquid 변환 작업을 수행하여, 모든 웹 서버에 곧바로 게시할 수 있는 완성된 정적 웹사이트를 만들어냅니다. 또한 Jekyll 은 GitHub Pages 의 내부 엔진으로 사용되고 있습니다. 이 말은, Jekyll 을 사용하여 자신의 프로젝트 페이지나 블로그, 웹사이트를 GitHub 에 무료로 호스팅 할 수 있다는 뜻입니다.
