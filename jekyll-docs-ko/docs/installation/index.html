설치

단 몇 분만에 Jekyll 설치와 실행 준비를 끝마칠 수 있습니다. 만약 골치 아픈 일이 발생한다면, 문제점에 대한 설명과 개선 방안을 이슈로 등록 (또는 Pull Request 를 제출) 해 주시기 바랍니다.

준비물

Jekyll 설치는 쉽고 간단하지만, 필요한 것들이 모두 준비되어 있는지 먼저 확인해보세요.

  • 루비
  • RubyGems
  • 리눅스, 유닉스, 또는 맥 OS X
윈도우즈에서 Jekyll 실행하기

윈도우즈용 Jekyll 이 존재하지만, 공식 문서에서는 윈도우즈 환경에 대한 내용을 다루지 않습니다.

RubyGems 으로 설치하기

Jekyll 을 설치하는 가장 좋은 방법은 RubyGems 을 사용하는 것입니다. 터미널 프롬프트에서, 이 명령을 실행합니다:

$ gem install jekyll

이 명령으로 Jekyll 의 모든 Gem 의존요소들이 자동으로 설치되기 때문에, 신경쓸 것은 전혀 없습니다. Jekyll 설치에 문제가 발생한다면, 문제해결 페이지를 확인해보거나 Jekyll 커뮤니티가 해당 문제를 개선할 수 있도록 이슈로 등록해 주시기 바랍니다.

Xcode Command-Line Tools 설치하기

맥 OS X 환경에서 Native extension 을 사용하는 Jekyll 의 의존요소를 설치할 때 문제가 발생한다면, Xcode 와 Xcode 의 Command-Line Tools 를 설치해야 합니다. Preferences → Downloads → Components 에서 다운로드 하세요.

부가 기능

Jekyll 은 다양한 부가 기능을 지원하고 있습니다. 당신이 Jekyll 을 어떻게 사용할 것인가에 따라, 원하는 기능을 골라서 설치하면 됩니다. 예를 들면, LaTeX 와 같은 부가 기능을 설치하여 기본 컨텐츠 렌더링 엔진을 대신할 수 있습니다. 더 자세한 내용은 부가기능 페이지를 확인하세요.

ProTips™: 구문강조를 사용하세요

Jekyll 을 사용하는 당신과 같은 사람은, Pygments 를 사용한 구문강조 기능을 사용할 기회가 있을 것입니다. 이 기능이 필요하다면, 먼저 활성화 방법을 꼭 확인해보세요.

이제 필요한 것들이 모두 설치되었습니다. 시작해 봅시다!
