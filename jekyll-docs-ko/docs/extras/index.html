부가기능

Jekyll 은 다양한 부가기능(선택사항)을 지원합니다. Jekyll 을 어떻게 사용할 것인지에 따라 원하는 기능을 골라서 설치하면 됩니다.

LaTeX 지원

Maruku 에 PNG 렌더링을 위한 LaTeX 지원을 추가할 수 있습니다. blahtex (버전 0.6) 가 필요한데, dvips 와 함께 $PATH 에 등록해야 합니다. Maruku 에 사용되는 dvips 경로를 바꿔야 한다면, Remi’s Maruku fork 를 살펴보세요.

RDiscount

마크다운으로 Maruku 대신 RDiscount 를 사용하려고 한다면, 먼저 설치가 필요합니다:

$ [sudo] gem install rdiscount

다음, Jekyll 실행 옵션에 반영되도록 _config.yml 파일에 RDiscount 를 마크다운 엔진으로 지정합니다.

# _config.yml 파일 내용
markdown: rdiscount

Kramdown

또, Maruku 대신 Kramdown 을 사용할 수 있습니다. Kramdown 을 설치하세요:

$ [sudo] gem install kramdown

다음 _config.yml 파일에 Kramdown 을 마크다운 엔진으로 지정합니다.

# _config.yml 파일 내용
markdown: kramdown

Kramdown 에는 HTML 결과를 조정할 수 있는 다양한 옵션이 있습니다. 환경설정 페이지에 Jekyll 이 사용하는 기본값들의 목록이 있습니다. Kramdown website 에서는 전체 옵션 목록을 찾을 수 있습니다.
