빠른 시작 안내

서둘러 사용해보고자 하는 분들을 위해, Jekyll 사이트를 구동하는 가장 간단한 방법을 알려드립니다.

~ $ gem install jekyll
~ $ jekyll new myblog
~ $ cd myblog
~/myblog $ jekyll serve
# => 브라우저로 http://localhost:4000 을 열어봅니다

여기까진 별로 특별한 것도 없습니다. 하지만, 블로그 포스트를 생성하거나 템플릿과 레이아웃을 다룰 때, 머리말을 사용하거나 Jekyll 의 다양한 환경설정 옵션들을 활용할 때 진짜 마술같은 일이 벌어집니다.

Redcarpet 을 기본 Markdown 엔진으로 사용합니다

Jekyll 1.1 버전부터, jekyll new 명령으로 생성되는 사이트의 기본 Markdown 엔진이 Redcarpet 으로 변경되었습니다.

문제가 발생한다면, 필요한 것들이 모두 올바르게 준비되었는지 확인해보세요.
